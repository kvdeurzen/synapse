# Milestone v1.0: Data Layer

**Status:** SHIPPED 2026-03-01
**Phases:** 1-9
**Total Plans:** 24

## Overview

Synapse v1.0 delivers the complete data layer: an MCP server with 18 tools for storing, querying, and searching project knowledge and source code via LanceDB with semantic/hybrid search. Agents can store documents with versioning, index codebases with AST-aware symbol extraction, search via semantic/fulltext/hybrid modes, and assemble token-budget-aware context from both documents and code.

## Milestone Summary

**Key Decisions:**
- LanceDB embedded over Postgres/SQLite+pgvector (zero-config, vector + FTS + SQL in one)
- Chunk at write time, not query time (better embedding quality at ~512 tokens)
- Fail-fast on missing Ollama (mixed embedding spaces fracture vector search)
- Versioning via superseded rows (full history preserved, append-friendly)
- RRF for hybrid search (simple, no score normalization needed)
- Two-phase get_smart_context (agent controls what it loads)
- tree-sitter for code parsing (fast, incremental, multi-language)
- Separate code_chunks table (different schema, independent indexing lifecycle)
- Auto relationships from AST imports (knowledge graph stays fresh)
- Include v2 schema foundations in v1 (parent_id, depth, decision_type — cheaper than migration later)

**Issues Resolved:**
- INT-01: project_meta never seeded, causing last_index_at permanently null (Phase 8)
- Flow 6: Incremental re-index stale detection incomplete (Phase 8)
- 5 documentation accuracy issues (Phase 9)

**Issues Deferred:**
- None — all identified gaps closed by Phases 8-9

**Technical Debt Incurred:**
- escapeSQL helper duplicated in init-project.ts and index-codebase.ts (shared module extraction deferred)
- index_codebase overwrites project_meta.created_at on each run (no consumer reads this field)
- INT-02: AST import edges use file path IDs, unreachable from get_related_documents which queries by ULID doc_id
- Missing requirements-completed frontmatter in several phase SUMMARYs (documentation convention gap)
