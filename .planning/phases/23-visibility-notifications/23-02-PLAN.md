<objective>
Enhance the project_overview MCP tool to return task tree progress, pool status, and blocked item details, then update /synapse:status to consume the enriched response and render a "Needs Your Attention" section.

Purpose: Currently project_overview only returns document statistics. Users need to make multiple MCP calls to get a complete picture. This plan makes a single project_overview call return everything needed for a comprehensive dashboard — per-epic progress, RPEV pipeline status, pool activity, and items requiring attention.

Output: Enhanced ProjectOverviewResult interface with task_progress, pool_status, and needs_attention sections. Updated /synapse:status command template rendering the full dashboard from a single project_overview call.
</objective>
