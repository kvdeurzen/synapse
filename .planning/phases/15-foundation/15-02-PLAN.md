resolveConfig API (from 15-RESEARCH.md):
```javascript
// packages/framework/hooks/lib/resolve-config.js
// Returns absolute path to config file, or null if not found
// Search order: walk-up .synapse/config/ -> monorepo packages/framework/config/ fallback
export function resolveConfig(filename: string): string | null;
```

Usage pattern for fail-closed hooks (tier-gate, tool-allowlist):
```javascript
import { resolveConfig } from './lib/resolve-config.js';

const configPath = resolveConfig('trust.toml');
if (!configPath) {
  process.stdout.write(denyOutput('DENIED: trust.toml not found. Denying as fail-closed precaution.'));
  process.exit(0);
}
const config = parse(fs.readFileSync(configPath, 'utf8'));
```

Usage pattern for audit-log (derive project root from resolved path):
```javascript
const synapseConfigPath = resolveConfig('project.toml');
const projectRoot = synapseConfigPath
  ? path.dirname(path.dirname(path.dirname(synapseConfigPath)))
  : (process.env.CLAUDE_PROJECT_DIR || process.cwd());
const logPath = path.join(projectRoot, '.synapse-audit.log');
```
