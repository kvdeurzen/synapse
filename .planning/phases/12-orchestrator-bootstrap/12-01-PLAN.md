  <action>
    1. Create `src/config.ts` with the following structure:

       a. Shared `loadAndValidate<T>(filePath, schema, options?)` helper:
          - `options.optional`: if true, return default value when file missing (for secrets.toml)
          - Read file with readFileSync. On ENOENT: if optional, return schema default; otherwise print clear error: `[synapse-framework] ${filePath} not found. Create it from ${filePath}.template or run setup.` and throw ConfigError
          - Parse TOML with smol-toml `parse()`. On parse error: print `[synapse-framework] Malformed ${filePath}: ${err.message}` and throw ConfigError
          - Validate with Zod `safeParse()`. On validation error: collect ALL issues, format as bullet list, print `[synapse-framework] Configuration error(s) in ${filePath}:\n${errors}` and throw ConfigError
          - Return typed result

       b. `SynapseFrameworkConfigSchema` Zod schema:
          ```typescript
          const SynapseFrameworkConfigSchema = z.object({
            server: z.object({
              db: z.string().min(1, "db path is required"),
              ollama_url: z.string().url().default("http://localhost:11434"),
              embed_model: z.string().default("nomic-embed-text"),
            }),
            connection: z.object({
              transport: z.enum(["stdio"]).default("stdio"),
              command: z.string().default("bun"),
              args: z.array(z.string()).default([]),
            }).default({}),
          });
          ```

       c. `TrustConfigSchema` Zod schema:
          ```typescript
          const autonomyLevel = z.enum(["autopilot", "co-pilot", "advisory"]);
          const TrustConfigSchema = z.object({
            domains: z.record(z.string(), autonomyLevel).default({}),
            approval: z.object({
              decomposition: z.enum(["always", "strategic", "none"]).default("strategic"),
            }).default({}),
          });
          ```

       d. `AgentsConfigSchema` Zod schema:
          ```typescript
          const AgentsConfigSchema = z.object({
            agents: z.record(z.string(), z.object({
              model: z.enum(["opus", "sonnet"]).default("sonnet"),
              tier: z.number().int().min(0).max(3).optional(),
              skills: z.array(z.string()).default([]),
            })).default({}),
          });
          ```

       e. `SecretsConfigSchema` Zod schema:
          ```typescript
          const SecretsConfigSchema = z.object({
            anthropic_api_key: z.string().optional(),
          }).passthrough(); // Allow arbitrary keys for future secrets
          ```

       f. Export loader functions:
          - `loadSynapseConfig(configPath?)` -- calls loadAndValidate with SynapseFrameworkConfigSchema
          - `loadTrustConfig(configPath?)` -- calls loadAndValidate with TrustConfigSchema
          - `loadAgentsConfig(configPath?)` -- calls loadAndValidate with AgentsConfigSchema
          - `loadSecretsConfig(configPath?)` -- calls loadAndValidate with SecretsConfigSchema, optional=true
          - `loadAllConfig(configDir?)` -- convenience function that loads all four configs from a directory

       g. Export `ConfigError` class extending Error for structured error handling in tests

       h. Export all schemas so tests and other modules can reference them
